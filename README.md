# NetflixIMDB

This project analyzes an open dataset from Kaggle containing Netflix movies and TV shows with IMDB ratings.

## Dataset
Source: Kaggle - Netflix IMDB Scores
Link: https://www.kaggle.com/datasets/thedevastator/netflix-imdb-scores?resource=download

## Analysis
The notebook explains the structure of the dataset, checks for missing values,
creates summary statistics, and builds graphs that show the differences between movies and TV shows.

## Files
- Analysis.ipynb – main notebook
- Analysis.html – rendered version
- Analysis.md – markdown version
